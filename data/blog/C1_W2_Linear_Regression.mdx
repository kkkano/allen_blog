<a name="2.3"></a>
### 2.3 线性回归基础回顾

线性回归的目标是通过一个线性函数拟合数据。在这个任务中，我们将调整参数 \(w\) 和 \(b\)，使模型 \(f_{w,b}(x) = wx + b\) 尽可能贴近数据集。

- **模型函数**：\(f_{w,b}(x) = wx + b\)，其中 \(x\) 是城市人口，\(f_{w,b}(x)\) 是预测的利润。
- **训练目标**：找到最佳的 \(w\) 和 \(b\)，使预测值与实际值尽可能接近。
- **成本函数**：我们用 \(J(w,b) = \frac{1}{2m} \sum_{i=0}^{m-1} (f_{w,b}(x^{(i)}) - y^{(i)})^2\) 来衡量模型的误差，\(m\) 是样本数。
- **优化方法**：通过**梯度下降法**找到使 \(J(w,b)\) 最小的 \(w\) 和 \(b\)。

训练完成后，模型就能根据输入的城市人口预测餐厅的月利润。

<a name="2.4"></a>
### 2.4 计算成本函数

梯度下降需要不断调整 \(w\) 和 \(b\)，以减小成本 \(J(w,b)\)。为了监控优化过程，我们需要一个函数来计算成本。成本函数的定义是：

\[ J(w,b) = \frac{1}{2m} \sum_{i=0}^{m-1} (f_{w,b}(x^{(i)}) - y^{(i)})^2 \]

- \(f_{w,b}(x^{(i)}) = wx^{(i)} + b\) 是模型预测值。
- \(y^{(i)}\) 是实际利润。
- \(m\) 是训练样本数。

<a name="ex01"></a>
#### 练习 1：实现成本函数

请完成以下 `compute_cost` 函数，计算成本 \(J(w,b)\)：

```python
def compute_cost(x, y, w, b):
    """
    计算线性回归的成本函数。
    
    参数:
        x (ndarray): 形状 (m,)，模型输入（城市人口）
        y (ndarray): 形状 (m,)，标签（实际利润）
        w, b (scalar): 模型参数
    
    返回:
        total_cost (float): 使用 w, b 拟合数据的成本
    """
    # 训练样本数
    m = x.shape[0]
    
    total_cost = 0
    
    ### 开始你的代码 ###
    cost_sum = 0
    for i in range(m):
        f_wb = w * x[i] + b  # 预测值
        cost = (f_wb - y[i]) ** 2  # 单样本误差
        cost_sum += cost  # 累加误差
    total_cost = (1 / (2 * m)) * cost_sum  # 计算平均成本
    ### 结束你的代码 ###
    
    return total_cost
```

<a name="2.5"></a>
### 2.5 梯度下降法

梯度下降是一种迭代优化算法，通过以下步骤更新参数：

\[ \begin{align*} & \text{重复直到收敛:} \; \lbrace \newline & \; b := b - \alpha \frac{\partial J(w,b)}{\partial b} \newline & \; w := w - \alpha \frac{\partial J(w,b)}{\partial w} \newline & \rbrace \end{align*} \]

其中：
- \(\frac{\partial J(w,b)}{\partial b} = \frac{1}{m} \sum_{i=0}^{m-1} (f_{w,b}(x^{(i)}) - y^{(i)})\)
- \(\frac{\partial J(w,b)}{\partial w} = \frac{1}{m} \sum_{i=0}^{m-1} (f_{w,b}(x^{(i)}) - y^{(i)})x^{(i)}\)
- \(\alpha\) 是学习率，控制每次更新的步长。

<a name="ex02"></a>
#### 练习 2：实现梯度计算

请完成以下 `compute_gradient` 函数，计算梯度：

```python
def compute_gradient(x, y, w, b):
    """
    计算线性回归的梯度。
    
    参数:
        x (ndarray): 形状 (m,)，模型输入（城市人口）
        y (ndarray): 形状 (m,)，标签（实际利润）
        w, b (scalar): 模型参数
    
    返回:
        dj_dw (scalar): 成本对 w 的梯度
        dj_db (scalar): 成本对 b 的梯度
    """
    m = x.shape[0]
    dj_dw = 0
    dj_db = 0
    
    ### 开始你的代码 ###
    for i in range(m):
        f_wb = w * x[i] + b  # 预测值
        dj_dw_i = (f_wb - y[i]) * x[i]  # w 的梯度
        dj_db_i = f_wb - y[i]  # b 的梯度
        dj_dw += dj_dw_i
        dj_db += dj_db_i
    dj_dw = dj_dw / m
    dj_db = dj_db / m
    ### 结束你的代码 ###
    
    return dj_dw, dj_db
```

<a name="2.6"></a>
### 2.6 使用批量梯度下降优化参数

现在，我们将使用批量梯度下降来找到最佳参数。以下是实现代码（无需修改，直接运行）：

```python
def gradient_descent(x, y, w_in, b_in, cost_function, gradient_function, alpha, num_iters):
    """
    执行批量梯度下降以优化参数。
    
    参数:
        x (ndarray): 形状 (m,)
        y (ndarray): 形状 (m,)
        w_in, b_in (scalar): 参数初始值
        cost_function: 计算成本的函数
        gradient_function: 计算梯度的函数
        alpha (float): 学习率
        num_iters (int): 迭代次数
    
    返回:
        w, b: 优化后的参数
        J_history, w_history: 成本和 w 的历史记录
    """
    m = len(x)
    J_history = []
    w_history = []
    w = copy.deepcopy(w_in)
    b = b_in
    
    for i in range(num_iters):
        dj_dw, dj_db = gradient_function(x, y, w, b)
        w = w - alpha * dj_dw
        b = b - alpha * dj_db
        
        if i < 100000:
            cost = cost_function(x, y, w, b)
            J_history.append(cost)
        
        if i % math.ceil(num_iters / 10) == 0:
            w_history.append(w)
            print(f"迭代 {i:4}: 成本 {float(J_history[-1]):8.2f}")
    
    return w, b, J_history, w_history
```

运行梯度下降：

```python

                              compute_cost, compute_gradient, alpha, iterations)

```

