<a name="2.3"></a>
### 2.3 线性回归基础

线性回归通过函数 `f(x) = w * x + b` 拟合数据：
- `x` 是人口，`f(x)` 是预测利润。
- 目标：调整 `w` 和 `b`，使预测接近实际值。
- 成本函数：`J(w,b)` 衡量误差，公式为平均平方差。
- 方法：用梯度下降法最小化 `J(w,b)`。

训练后，模型可预测新城市的利润。

---

<a name="2.4"></a>
### 2.4 计算成本函数

成本函数衡量模型误差，定义为平均平方差。我们需要一个函数计算它。

<a name="ex01"></a>
#### 练习 1：实现成本函数

完成以下 `compute_cost` 函数：

```python
def compute_cost(x, y, w, b):
    """
    计算线性回归成本。
    
    参数:
        x (ndarray): 形状 (m,)，输入（人口）
        y (ndarray): 形状 (m,)，标签（利润）
        w, b (scalar): 模型参数
    
    返回:
        total_cost (float): 成本值
    """
    m = x.shape[0]
    total_cost = 0
    
    ### 开始代码 ###
    cost_sum = 0
    for i in range(m):
        f_wb = w * x[i] + b
        cost = (f_wb - y[i]) ** 2
        cost_sum += cost
    total_cost = (1 / (2 * m)) * cost_sum
    ### 结束代码 ###
    
    return total_cost
```

测试：

```python
initial_w = 2
initial_b = 1
cost = compute_cost(x_train, y_train, initial_w, initial_b)
print(f'初始 w 时成本: {cost:.3f}')
```

**预期输出：**
```

```

<a name="2.5"></a>
### 2.5 梯度下降法

梯度下降通过迭代更新 `w` 和 `b` 减小成本：
- 对 `b` 的偏导：平均预测误差。
- 对 `w` 的偏导：平均预测误差乘以 `x`。
- 更新：`w` 和 `b` 减去学习率 `alpha` 乘以偏导。

<a name="ex02"></a>
#### 练习 2：实现梯度计算

完成以下 `compute_gradient` 函数：

```python
def compute_gradient(x, y, w, b):
    """
    计算梯度。
    
    参数:
        x (ndarray): 形状 (m,)，输入（人口）
        y (ndarray): 形状 (m,)，标签（利润）
        w, b (scalar): 参数
    
    返回:
        dj_dw (scalar): w 的梯度
        dj_db (scalar): b 的梯度
    """
    m = x.shape[0]
    dj_dw = 0
    dj_db = 0
    
    ### 开始代码 ###
    for i in range(m):
        f_wb = w * x[i] + b
        dj_dw_i = (f_wb - y[i]) * x[i]
        dj_db_i = f_wb - y[i]
        dj_dw += dj_dw_i
        dj_db += dj_db_i
    dj_dw = dj_dw / m
    dj_db = dj_db / m
    ### 结束代码 ###
    
    return dj_dw, dj_db
```

测试：

```python
initial_w = 0
initial_b = 0
dj_dw, dj_db = compute_gradient(x_train, y_train, initial_w, initial_b)
print('初始梯度:', dj_dw, dj_db)
```

**预期输出：**
```

```

<a name="2.6"></a>
### 2.6 优化参数

使用梯度下降优化参数：

```python
def gradient_descent(x, y, w_in, b_in, cost_function, gradient_function, alpha, num_iters):
    """
    执行梯度下降。
    
    参数:
        x, y (ndarray): 数据
        w_in, b_in (scalar): 初始参数
        cost_function: 计算成本函数
        gradient_function: 计算梯度函数
        alpha (float): 学习率
        num_iters (int): 迭代次数
    
    返回:
        w, b: 优化参数
    """
    w = w_in
    b = b_in
    
    for i in range(num_iters):
        dj_dw, dj_db = gradient_function(x, y, w, b)
        w = w - alpha * dj_dw
        b = b - alpha * dj_db
        
        if i % 150 == 0:
            cost = cost_function(x, y, w, b)
            print(f"迭代 {i:4}: 成本 {cost:.2f}")
    
    return w, b
```

运行：

```python
initial_w = 0.
initial_b = 0.
iterations = 1500
alpha = 0.01

w, b = gradient_descent(x_train, y_train, initial_w, initial_b,
                        compute_cost, compute_gradient, alpha, iterations)
print("优化后的 w, b:", w, b)
```

**预期输出：**
```

迭代  150: 成本 5.31
...
优化后的 w, b: 1.16636235 -3.63029143940436
```
