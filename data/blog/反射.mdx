### <font style="color:rgb(79, 79, 79);">反射简介</font>
#### <font style="color:rgb(79, 79, 79);">什么是反射</font>
[<font style="color:rgb(77, 77, 77);">JAVA反射机制</font>](https://so.csdn.net/so/search?q=JAVA%E5%8F%8D%E5%B0%84%E6%9C%BA%E5%88%B6&spm=1001.2101.3001.7020)<font style="color:rgb(77, 77, 77);">是</font>**<font style="color:red;">在运行状态中</font>**<font style="color:rgb(77, 77, 77);">，对于任意一个类，都能够知道这个类的所有属性和方法；对于任意一个对象，都能够调用它的任意一个方法和属性；这种动态获取的信息以及动态调用对象的方法的功能称为java语言的</font>[<font style="color:rgb(77, 77, 77);">反射机制</font>](https://so.csdn.net/so/search?q=%E5%8F%8D%E5%B0%84%E6%9C%BA%E5%88%B6&spm=1001.2101.3001.7020)<font style="color:rgb(77, 77, 77);">。</font>

#### <font style="color:rgb(79, 79, 79);">java反射的作用</font>
<font style="color:rgb(77, 77, 77);">在运行时判断任意一个对象所属的类</font>

<font style="color:rgb(77, 77, 77);">在运行时构造任意一个类的对象</font>

<font style="color:rgb(77, 77, 77);">在运行时判断任意一个类所具有的成员变量和方法</font>

<font style="color:rgb(77, 77, 77);">在运行时调用任意一个对象的方法</font>

#### <font style="color:rgb(79, 79, 79);">反射机制有什么用</font>
<font style="color:rgb(77, 77, 77);">通过java语言中的反射机制可以操作字节码文件（可以读和修改字节码文件。）</font>

<font style="color:rgb(77, 77, 77);">通过反射机制可以操作代码片段。（class文件。）</font>

#### <font style="color:rgb(79, 79, 79);">哪里会用到反射机制</font>
<font style="color:rgb(77, 77, 77);">Web服务器中利用反射调用了Sevlet的服务方法。</font>

<font style="color:rgb(77, 77, 77);">模块化的开发，通过反射去调用对应的字节码</font>

<font style="color:rgb(77, 77, 77);">动态代理设计模式也采用了反射机制</font>

<font style="color:rgb(77, 77, 77);">Spring／Hibernate 等框架，也是利用</font>[<font style="color:rgb(77, 77, 77);">CGLIB</font>](https://so.csdn.net/so/search?q=CGLIB&spm=1001.2101.3001.7020)<font style="color:rgb(77, 77, 77);"> </font><font style="color:rgb(77, 77, 77);">反射机制才得以实现:</font>

1. <font style="color:rgb(77, 77, 77);">JDBC中，利用反射动态加载了数据库驱动程序。</font>

### <font style="color:rgb(79, 79, 79);">反射使用步骤（获取 Class 对象、调用对象方法）</font>
#### <font style="color:rgb(79, 79, 79);">获取 Class 对象的三种方法</font>
<font style="color:rgb(77, 77, 77);">1.通过new对象实现反射机制: 对象.getClass()</font>

<font style="color:rgb(77, 77, 77);">2.通过路径实现反射机制: Class.forName(“完整类名带包名”) 静态方法</font>

<font style="color:rgb(77, 77, 77);">例如：com.mysql.jdbc.Driver Driver类已经被加载到 jvm中，并且完成了类的初始化工作就行了</font>

<font style="color:rgb(77, 77, 77);">3.通过类名实现反射机制: 任何类型（类名）.class。 获取Class<?> clz 对象</font>

##### <font style="color:rgb(79, 79, 79);">代码示例</font>
```plain
public class Student {
	private int id;
	String name;
 	protected boolean sex;
	public float score;
}

public class Get {
	//获取反射机制三种方式
	public static void main(String[] args) throws ClassNotFoundException {
        //方式一(通过建立对象)
        Student stu = new Student();
        Class classobj1 = stu.getClass();
        System.out.println(classobj1.getName());

        //方式二（所在通过路径-相对路径）
        Class classobj2 = Class.forName("fanshe.Student");
        System.out.println(classobj2.getName());

        //方式三（通过类名）
        Class classobj3 = Student.class;
        System.out.println(classobj3.getName());
	}
}


```

```plain
class ReflectTest02{
    public static void main(String[] args) throws ClassNotFoundException, InstantiationException, IllegalAccessException {

        // 通过反射机制，获取Class，通过Class来实例化对象

    }
}


```

##### <font style="color:rgb(79, 79, 79);">反编译一个类的属性Field（本类的所有属性，不包含父类和接口）, 代码示例</font>
```plain
//通过反射机制，反编译一个类的属性Field（了解一下）
class ReflectTest06{
    public static void main(String[] args) throws ClassNotFoundException {
		Class studentClass = Class.forName("javase.reflectBean.Student");
		Modifier.toString(studentClass.getModifiers();
		// Class类的getName方法
		studentClass.getSimpleName()
    
        //获取所有的属性
		Field[] declaredFields = clazz1.getDeclaredFields();
        for (int i = 0; i < declaredFields.length; i++) {
            //属性名称
            String fieldName = declaredFields[i].getName();
            //属性的类型 
            // declaredFields[i]..getType().getSimpleName()
            String fieldType = declaredFields[i].getType().getName();
            //获取属性的修饰符列表，返回的修饰符是一个数字，每个数字是修饰符的代号
            //用Modifier类的toString转换成字符串
            String fieldModifier = Modifier.toString(declaredFields[i].getModifiers());
            System.out.println(fieldModifier + " " + fieldType + " " + fieldName + ";");
        }
    }
}


```

##### <font style="color:rgb(79, 79, 79);">通过反射机制访问一个java对象的属性,代码示例</font>
```plain
/*
必须掌握：
怎么通过反射机制访问一个java对象的属性？
给属性赋值set
获取属性的值get
 */
class ReflectTest07{
    public static void main(String[] args) throws ClassNotFoundException, InstantiationException, IllegalAccessException, NoSuchFieldException {
        // 不使用反射机制给属性赋值
        Student student = new Student();
        // 给属性赋值三要素：给s对象的no属性赋值1111
        student.no = 1111;
        // 读属性值两个要素：获取s对象的no属性的值
        System.out.println(student.no);

        // 使用反射机制给属性赋值
        Class studentClass = Class.forName("javase.reflectBean.Student");

		// obj就是Student对象。（底层调用无参数构造方法）
        Object obj = studentClass.newInstance();
```

##### <font style="color:rgb(79, 79, 79);">通过反射修改数组的大小</font>
<font style="color:rgb(77, 77, 77);">原理：其本质就是通过反射得到原数组的类型，然后通过Array.newInstance()方法根据数组类型创造出一个指定长度的新数组，最后使用System.arraycopy()方法将原数组的值拷贝到新数组中</font>

```plain
public static Object arrayInc(Object obj, int length) {
  //取得传入数组的类型，以便于创造出同类型的数组
  Class<?> componentType = obj.getClass().getComponentType();
  //根据传入的数组类型创建出新的指定长度的数组实例
  Object newArr = Array.newInstance(componentType, length);
  //原有的数组长度
  int originArrLen = Array.getLength(obj);
  //将原有的数组数据拷贝到新的数组中去
  System.arraycopy(obj,0,newArr,0,originArrLen);
  //返回修改大小后的数组
  return newArr;
}
```
